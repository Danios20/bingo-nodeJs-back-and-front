# bingo-nodeJs-back-and-front
## Instalación
###Clona
Repositorio
```
https://github.com/Danios20/bingo-nodeJs-back-and-front.git
```
Pedimos a NPM que descargue las dependencias necesarias en node_modules/
```
$ npm install
```
## Iniciar App 
Corremos nuestro servidor de Express Js.
```
$ npm install
```
